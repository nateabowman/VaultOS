# Changelog

All notable changes to VaultOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete icon theme system with FreeDesktop structure
- Wallpaper and graphics guidelines and directory structure
- Enhanced GTK/Qt themes with additional widget coverage
- Application themes for Firefox, VS Code, MPV, file managers
- Enhanced VaultWM with workspaces, navigation, multiple layouts
- Shell utilities (pipboy-status, pipboy-viewer, theme-switcher)
- RPM repository management scripts
- CI/CD automation with GitHub Actions
- Comprehensive documentation (user guide, FAQ)
- Build automation scripts

### Changed
- Improved VaultWM status bar with system information
- Enhanced GTK theme with dark mode support and icon integration
- Expanded shell aliases and utilities
- Updated window manager with gap support and better tiling

### Fixed
- Various code improvements and bug fixes

## [1.0.0] - 2024-01-01

### Added
- Initial release of VaultOS
- GRUB and Plymouth boot themes
- VaultWM window manager (basic)
- GTK3/GTK4 and Qt themes
- Terminal themes and profiles
- Shell configurations (bash/zsh)
- Basic package structure
- Installation scripts
- Initial documentation
