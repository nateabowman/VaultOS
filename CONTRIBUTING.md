# Contributing to VaultOS

Thank you for your interest in contributing to VaultOS!

## How to Contribute

### Reporting Bugs
- Use the bug report template
- Include system information
- Provide steps to reproduce

### Suggesting Features
- Use the feature request template
- Explain the use case
- Propose a solution

### Submitting Themes
- Use the theme submission template
- Include screenshots
- Provide all theme files

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Code Style

- C code: Linux kernel style
- Shell scripts: Bash, use shellcheck
- Configuration: Follow existing format

## Testing

- Test in virtual machine
- Verify all components work
- Check for regressions
