# VaultOS Security Package

Security hardening scripts and configurations for VaultOS.

## Components

- **hardening.sh**: Post-installation security hardening script
- Firewall configurations
- Kernel security parameters
- Automatic security updates
- SSH security settings

## Usage

The hardening script runs automatically on package installation. To run manually:

```bash
sudo /usr/share/vaultos/scripts/hardening.sh
```

## Features

- Firewall configuration
- Kernel security parameters
- Automatic security updates
- SSH security hardening
- Service management
- File permission security
