# VaultOS SDK Examples

Example code and templates for VaultOS development.

## Examples

### Theme Template
`theme-template.css` - CSS template for creating custom themes

### Plugin Examples
- `example-statusbar-plugin.sh` - Example status bar plugin

## Usage

### Creating a Theme

1. Copy theme template:
   ```bash
   cp sdk/examples/theme-template.css my-theme.css
   ```

2. Customize colors and styles

3. Validate theme:
   ```bash
   vaultos-theme-validator my-theme/
   ```

### Creating a Plugin

1. Copy plugin example:
   ```bash
   cp sdk/plugins/example-statusbar-plugin.sh my-plugin.sh
   ```

2. Modify plugin code

3. Add plugin metadata

4. Test plugin
