# VaultOS Customization Tools

Advanced customization interfaces for themes and wallpapers.

## Tools

### theme-editor.sh
Interactive theme customization tool.

Usage:
```bash
vaultos-theme-editor
```

Features:
- Edit color schemes
- Preview themes
- Export/import themes
- Reset to defaults

### wallpaper-manager.sh
Wallpaper management tool.

Usage:
```bash
vaultos-wallpaper-manager
```

Features:
- List available wallpapers
- Set wallpaper
- Import wallpapers
- Configure slideshow

## Theme Storage

Custom themes are stored in:
- `~/.local/share/vaultos/themes/` - User themes

Themes can be exported and shared with others.

## Wallpaper Storage

Wallpapers are stored in:
- `/usr/share/vaultos/wallpapers/` - System wallpapers
- `~/.local/share/vaultos/wallpapers/` - User wallpapers
