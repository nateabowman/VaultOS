# VaultOS Diagnostics Tools

System diagnostic and troubleshooting utilities.

## Tools

### system-health.sh
Quick system health check.

Usage:
```bash
vaultos-system-health
```

Checks:
- Disk space usage
- Memory usage
- Network connectivity
- System load
- Service status
- File system errors

### troubleshoot.sh
Interactive troubleshooting wizard.

Usage:
```bash
vaultos-troubleshoot
```

Covers:
- Window manager issues
- Theme problems
- Display issues
- Audio problems
- Network issues
- Performance problems

## System Information

Gather system information:
```bash
# Hardware detection
vaultos-hardware-detect

# System status
pipboy-status

# Detailed system info
vaultos-system-health
```

## Problem Reporting

When reporting issues, include:
- System information (from hardware-detect)
- Health check output
- Relevant logs
- Steps to reproduce
