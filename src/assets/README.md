# VaultOS Assets

Wallpapers, graphics, and visual assets for VaultOS.

## Directory Structure

```
assets/
├── wallpapers/           # Desktop wallpapers
│   ├── pipboy/          # Pip-Boy green themed wallpapers
│   ├── vaulttec/        # Vault-Tec blue/gold themed wallpapers
│   └── patterns/        # Retro-futuristic patterns
├── boot/                # Boot graphics (GRUB, Plymouth)
│   ├── grub/            # GRUB bootloader graphics
│   └── plymouth/        # Plymouth boot splash graphics
├── logos/               # VaultOS and Vault-Tec logos
└── misc/                # Miscellaneous graphics
```

## Wallpapers

Place wallpapers in `wallpapers/` directory:
- Pip-Boy interface backgrounds
- Vault door images
- Retro-futuristic patterns
- Fallout-themed scenes

Recommended resolutions:
- 1920x1080 (Full HD)
- 2560x1440 (QHD)
- 3840x2160 (4K UHD)

## Boot Graphics

Graphics for boot system:
- GRUB background images
- Plymouth splash graphics
- Boot animation frames

## Logos

- VaultOS logo
- Vault-Tec logo (inspired)
- Pip-Boy graphics
- Branding elements

## Asset Guidelines

### Colors
- Primary: #00FF41 (Pip-Boy green)
- Background: #000000 (Black)
- Accent: #003300 (Dark green)
- Secondary: #1E3A8A (Vault-Tec blue)
- Gold: #FFD700 (Vault-Tec gold)

### Style
- Retro-futuristic
- 1950s atomic age aesthetic
- Clean, minimal design
- High contrast for readability

### Formats
- Raster: PNG (with transparency), JPG
- Vector: SVG
- Animation: GIF, PNG sequences

## Usage

Assets are referenced in:
- GRUB theme configuration
- Plymouth theme scripts
- Window manager
- Desktop backgrounds
- Application themes
