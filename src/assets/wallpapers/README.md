# VaultOS Wallpapers

Fallout-themed desktop wallpapers for VaultOS.

## Wallpaper Themes

### Pip-Boy Interface
- Green monochrome Pip-Boy screen
- Terminal-style interface
- Retro-futuristic design

### Vault-Tec
- Vault door imagery
- Blue and gold color scheme
- Corporate aesthetic

### Wasteland
- Post-apocalyptic scenes
- Desert landscapes
- Ruined buildings

## Specifications

- **Formats**: PNG, JPG
- **Resolutions**: 1920x1080, 2560x1440, 3840x2160
- **Aspect Ratio**: 16:9 (primary), 21:9 (ultrawide)
- **Color Depth**: 24-bit or 32-bit (with alpha)

## Installation

Wallpapers will be installed to `/usr/share/backgrounds/vaultos/` during system installation.

To set a wallpaper manually:
```bash
# For GNOME
gsettings set org.gnome.desktop.background picture-uri file:///usr/share/backgrounds/vaultos/wallpaper.png

# For other DEs/WMs
# Use your desktop environment's wallpaper settings
```

## Creating Wallpapers

Tools:
- GIMP
- Inkscape
- Blender (for 3D renders)
- ImageMagick (for processing)

Color palette:
- Pip-Boy green: #00FF41
- Black: #000000
- Dark green: #003300
- Vault-Tec blue: #1E3A8A
- Vault-Tec gold: #FFD700
