# Vault-Tec Wallpapers

Wallpapers featuring Vault-Tec blue (#1E3A8A) and gold (#FFD700) aesthetic.

## Required Files

Place wallpapers in this directory with the following naming:
- `vaulttec-{description}-{resolution}.png`
- Example: `vaulttec-door-1920x1080.png`

## Resolutions

Each wallpaper should be provided in:
- 1920x1080 (Full HD)
- 2560x1440 (QHD)
- 3840x2160 (4K UHD)

See `WALLPAPER_GUIDELINES.md` in parent directory for complete specifications.
