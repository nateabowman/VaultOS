# Pip-Boy Wallpapers

Wallpapers featuring Pip-Boy green (#00FF41) aesthetic.

## Required Files

Place wallpapers in this directory with the following naming:
- `pipboy-{description}-{resolution}.png`
- Example: `pipboy-grid-1920x1080.png`

## Resolutions

Each wallpaper should be provided in:
- 1920x1080 (Full HD)
- 2560x1440 (QHD)
- 3840x2160 (4K UHD)

See `WALLPAPER_GUIDELINES.md` in parent directory for complete specifications.
