# GRUB Boot Graphics

Graphics for the GRUB bootloader theme.

## Required Files

- `background.png` - Main background image (1920x1080 recommended)
- `terminal_box_*.png` - Terminal box graphics (optional)
- `select_*.png` - Selection highlight graphics (optional)

See `BOOT_GRAPHICS_GUIDELINES.md` in parent directory for complete specifications.

## Usage

These graphics are copied to `/boot/grub2/themes/vaultos/` during installation and referenced in `theme.txt`.
