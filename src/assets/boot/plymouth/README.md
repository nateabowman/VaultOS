# Plymouth Boot Splash Graphics

Graphics for the Plymouth boot splash theme.

## Required Files

- `vault-door.png` - Vault door image for animation
- `pipboy.png` - Pip-Boy image for animation
- `progress-bar.png` - Progress bar graphic (optional)
- Animation frame sequences: `frame-*.png` (if using frame-based animation)

See `BOOT_GRAPHICS_GUIDELINES.md` in parent directory for complete specifications.

## Usage

These graphics are copied to `/usr/share/plymouth/themes/vaultos/` during installation and referenced in `vaultos.script`.
