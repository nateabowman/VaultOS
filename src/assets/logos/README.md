# VaultOS Logos

Logos and branding elements for VaultOS.

## Directory Structure

Organize logos by type and format:
- `vaultos/` - Main VaultOS logos
- `vaulttec/` - Vault-Tec inspired logos
- `pipboy/` - Pip-Boy graphics and elements

## File Formats

- `svg/` - Vector source files (preferred)
- `png/` - Raster versions in various sizes
- `variants/` - Color and style variants

See `LOGO_GUIDELINES.md` for complete specifications and design guidelines.
